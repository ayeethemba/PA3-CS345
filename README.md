PA3 - Simple Pong Game
Themba's Pong is a single-player racquetball-style game built with p5.js.

Controls
KeyAction↑ Up ArrowMove paddle up↓ Down ArrowMove paddle down

How to Play

Use the arrow keys to move your paddle and bounce the ball back to the left
Every successful bounce earns +1 point
If the ball gets past your paddle, you lose -1 point and the ball relaunches
Click START on the title screen to begin


Project Requirements

✅ Ball moves diagonally across the screen
✅ Ball bounces off the top, bottom, and left walls
✅ Player controls a paddle on the right side of the screen using arrow keys
✅ Scoring system: +1 for a successful bounce, -1 for a miss
✅ Score is displayed at the top left of the screen
✅ Ball relaunches from the left side after a miss


Additional Features

🎵 Background Music — plays automatically when the game starts
🖥️ Start Screen — custom title screen with a galaxy background and glowing title
🚧 Paddle Boundary — the paddle cannot move off the top or bottom of the screen
